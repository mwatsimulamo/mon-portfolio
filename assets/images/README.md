# Dossier Images

Placez vos images dans ce dossier :

## Structure recommandée

- `profile.jpg` - Votre photo de profil (format carré recommandé, 300x300px minimum)
- `projects/` - Images de vos projets
  - `cardano-nft-marketplace.jpg`
  - `voting-platform.jpg`
  - `ecommerce.jpg`
  - etc.

## Formats supportés

- JPG / JPEG
- PNG
- WebP (recommandé pour de meilleures performances)

## Taille recommandée

- Photo de profil : 300x300px à 500x500px
- Images de projets : 800x600px (ratio 4:3) ou 1200x600px (ratio 16:9)

## Optimisation

Pour de meilleures performances, optimisez vos images avant de les ajouter :
- Utilisez des outils comme [TinyPNG](https://tinypng.com/) ou [Squoosh](https://squoosh.app/)
- Compressez les images sans perte de qualité visible
- Utilisez le format WebP si possible

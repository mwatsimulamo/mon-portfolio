# Dossier CV

Placez votre CV en format PDF dans ce dossier.

## Fichier requis

- `cv.pdf` - Votre CV au format PDF

## Recommandations

- Nommez le fichier exactement `cv.pdf`
- Assurez-vous que le PDF est optimisé (taille raisonnable)
- Vérifiez que le PDF est lisible et bien formaté
- Le CV sera téléchargeable depuis la section CV du portfolio

## Alternative

Si vous préférez un autre nom de fichier, modifiez les références dans :
- `index.html` (ligne avec `href="assets/cv/cv.pdf"`)
- `script.js` (variable `cvPath` dans la fonction `initCVPreview`)
